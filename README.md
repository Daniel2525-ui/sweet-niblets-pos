# Sweet Niblets POS System

A modern Point of Sale (POS) and Inventory Management System built to streamline sales transactions and inventory tracking for Sweet Niblets.

## Features

* Product Management

  * Add new products
  * Update product information
  * Delete products

* Inventory Management

  * Track available stock
  * Automatic stock deduction after sales
  * Monitor inventory levels

* Point of Sale

  * Fast and simple checkout process
  * Add products to cart
  * Calculate total sales automatically

* Sales Tracking

  * Record transactions
  * View sales history
  * Monitor business performance

* Responsive Design

  * Works on desktop and mobile devices

## Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* Local Storage

## Installation

1. Clone the repository

```bash
git clone https://github.com/Daniel2525-ui/sweet-niblets-pos.git
```

2. Navigate to the project folder

```bash
cd sweet-niblets-pos
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
